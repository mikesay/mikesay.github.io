GitеҠ еҜҶеӯҳеӮЁж–Үд»¶

жң¬ж–ҮжңҖеҗҺжӣҙж–°дәҺпјҡ2020е№ҙ12жңҲ29ж—Ҙ дёӢеҚҲ

дёҚиҰҒе°ҶеҢ…еҗ«з”ЁжҲ·еҗҚгҖҒеҜҶз ҒгҖҒAPIд»ӨзүҢ(Token)зӯүеҗ„зұ»ж•Ҹж„ҹдҝЎжҒҜж”ҫеңЁGitд»Јз Ғд»“еә“дёӯе·Із»ҸжҲҗдёәдәҶеӨ§е®¶зҡ„е…ұиҜҶпјҢе°Өе…¶жҳҜд»Јз Ғд»“еә“жүҳз®ЎеңЁGitHubгҖҒGitLabгҖҒBitbucketзӯүжҸҗдҫӣе…¬е…ұд»Јз Ғд»“еә“жңҚеҠЎзҡ„е№іеҸ°дёҠж—¶пјҢжӣҙйңҖиҰҒдё”еҖјеҫ—д»»дҪ•зҡ„д»Јд»·еҺ»е°ҪйҮҸйҒҝе…Қиҝҷз§ҚдәӢжғ…зҡ„еҸ‘з”ҹгҖӮжң¬ж–Үд»Ӣз»Қе№¶дҪҝз”Ёgit-secretе°Ҷж–Үд»¶еҠ еҜҶеҗҺеӯҳеӮЁеңЁGitд»Јз Ғд»“еә“дёӯпјҢд»ҘеҸҠеҹәдәҺgit-secretзҡ„еӨҡдәәеҚҸеҗҢе·ҘдҪңзҡ„жөҒзЁӢгҖӮ

з§Ғжңүд»“еә“жҳҜеҗҰе®үе…Ё

д№ҹи®ёжңүдәәдјҡиҜҙпјҡвҖқжҲ‘жҠҠд»Јз Ғд»“еә“и®ҫжҲҗз§Ғжңүзҡ„дёҚе°ұе®үе…ЁдәҶпјҹвҖңпјҢз„¶иҖҢжғ…еҶөе№¶йқһеҰӮжӯӨпјҢеӣ дёәеҚідҪҝжҳҜз§Ғжңүд»“еә“пјҢд№ҹеҗҢж ·йқўдёҙзқҖдёҚеҗҢзұ»еһӢзҡ„йЈҺйҷ©гҖӮеҪ“дҪ йӣҶжҲҗжҹҗдёӘз¬¬дёүж–№жңҚеҠЎеҲ°дҪ зҡ„з§Ғжңүд»“еә“пјҲжҜ”еҰӮйӣҶжҲҗз¬¬дёүж–№зҡ„жҢҒз»ӯйӣҶжҲҗжңҚеҠЎпјүж—¶пјҢдҪ зҡ„з§Ғжңүд»“еә“е°ұејҖж”ҫз»ҷдәҶз¬¬дёүж–№жңҚеҠЎпјҢиҝҷдәӣз¬¬дёүж–№жңҚеҠЎе°ұжңүеҸҜиғҪиҜ»еҸ–дҪ еӯҳеӮЁеңЁд»Јз Ғд»“еә“дёӯзҡ„ж•Ҹж„ҹдҝЎжҒҜгҖӮеҰӮжһңз¬¬дёүж–№жңҚеҠЎйҒӯеҸ—дәҶй»‘е®ўж”»еҮ»пјҢйӮЈд№Ҳй»‘е®ўе°ұеҸҜд»ҘйҖҡиҝҮиҝҷдёӘз¬¬дёүж–№жңҚеҠЎиҺ·еҸ–дҪ зҡ„Gitд»“еә“дёӯзҡ„ж•Ҹж„ҹдҝЎжҒҜгҖӮ

дёәд»Җд№ҲиҝҳиҰҒе°Ҷж•Ҹж„ҹдҝЎжҒҜж”ҫеңЁGitд»Јз Ғд»“еә“дёӯ

жӯЈеҰӮж–ҮеҲқи°ҲеҲ°зҡ„вҖқдёҚиҰҒе°Ҷж•Ҹж„ҹдҝЎжҒҜж”ҫеңЁGitд»Јз Ғд»“еә“дёӯе·ІжҳҜе…ұиҜҶвҖңпјҢйӮЈдёәд»Җд№Ҳиҝҳз”Ёи°Ҳе°Ҷж•Ҹж„ҹдҝЎжҒҜж”ҫеңЁGitд»Јз Ғд»“еә“дёӯпјҹжӯЈжүҖи°“ж— йЈҺдёҚиө·жөӘпјҢеӯҳеңЁеҚіеҗҲзҗҶгҖӮжҠӣејҖж•Ҹж„ҹдҝЎжҒҜиҝҷдёӘзү№еҫҒдёҚи°ҲпјҢеҢ…еҗ«иҝҷдәӣдҝЎжҒҜзҡ„ж–Үд»¶е…¶е®һе’ҢжҲ‘д»¬зҡ„д»Јз Ғж–Үд»¶жІЎд»Җд№ҲеҢәеҲ«пјҢеҰӮжһңдёҚж”ҫеңЁGitд»“еә“дёӯпјҢе°ұдёҚиғҪеҜ№е®ғд»¬еҒҡзүҲжң¬з®ЎзҗҶпјҢжҜ”еҰӮеҜ№ж–Үд»¶еҗҚе’Ңи·Ҝеҫ„зҡ„еҸҳжӣҙпјҢеҜҶз Ғзӯүж•Ҹж„ҹдҝЎжҒҜзҡ„еҸҳжӣҙпјҢж–°дҝЎжҒҜзҡ„ж·»еҠ зӯүйғҪжІЎжі•иҝҪиёӘе®ғд»¬зҡ„еҸҳжӣҙи®°еҪ•пјҢе°Өе…¶еңЁжһ„е»әдёҖдёӘиҮӘеҠЁйғЁзҪІзі»з»ҹж—¶пјҢдҪ е°ұдёҚеҫ—дёҚз»ҙжҠӨдёҖдёӘйўқеӨ–зҡ„е®үе…ЁжңҚеҠЎжқҘеҠ еҜҶеӯҳеӮЁиҝҷдәӣеҢ…еҗ«ж•Ҹж„ҹдҝЎжҒҜзҡ„й…ҚзҪ®ж–Үд»¶пјҢиҖҢиҮӘеҠЁйғЁзҪІзі»з»ҹд№ҹйңҖиҰҒи®ҫи®ЎйўқеӨ–зҡ„ж–№жі•е’ҢжӯҘйӘӨжқҘиҜ»еҸ–иҝҷдәӣй…ҚзҪ®ж–Үд»¶пјҢжӣҙдёәйә»зғҰзҡ„жҳҜиҮӘеҠЁйғЁзҪІзі»з»ҹж— жі•йҖҡиҝҮGitеҸҳжӣҙи®°еҪ•жқҘжҢҮе®ҡдҪҝз”Ёзҡ„й…ҚзҪ®ж–Үд»¶зҡ„зүҲжң¬гҖӮ

git-secretзҡ„еҺҹзҗҶ

ж—ўиҰҒе°ҶеҢ…еҗ«ж•Ҹж„ҹдҝЎжҒҜзҡ„й…ҚзҪ®ж–Үд»¶дҝқеӯҳеңЁGitд»Јз Ғд»“еә“дёӯпјҲиғҪеёҰжқҘеҫҲеӨҡдҫҝеҲ©пјүпјҢеҸҲиҰҒдҝқиҜҒе®үе…ЁпјҢйӮЈд№ҲжңҖзӣҙжҺҘзҡ„еҒҡжі•е°ұжҳҜе°Ҷиҝҷдәӣж–Үд»¶еҠ еҜҶеҗҺеҶҚдҝқеӯҳеҲ°Gitд»“еә“дёӯпјҢиҖҢgit-secretеҲҷж»Ўи¶ідәҶиҝҷдёҖйңҖжұӮпјҢеҗҢж—¶git-secretд№ҹиғҪдҝқиҜҒеӨҡдәәеҚҸеҗҢе·ҘдҪңгҖӮд»ҘдёӢжҳҜgit-secretжҸҗдҫӣзҡ„дёҖдәӣеҠҹиғҪпјҡ

  • еҲ©з”ЁGnuPGзҡ„еӨҡеҜҶй’ҘеҠ еҜҶеҠҹиғҪпјҡеӨҡдёӘе…¬й’ҘеҠ еҜҶж–Үд»¶пјҢдёҚеҗҢзҡ„з§Ғй’Ҙи§ЈеҜҶж–Үд»¶гҖӮиҝҷж ·пјҢе·ҘдҪңеңЁGitд»“еә“дёӯзҡ„ејҖеҸ‘дәәе‘ҳеҸӘйңҖиҰҒе°ҶиҮӘе·ұзҡ„е…¬й’ҘдҝқеӯҳеңЁд»Јз Ғд»“еә“дёӯз”ЁжқҘеҠ еҜҶж–Үд»¶пјҢеҗҢж—¶еңЁиҮӘе·ұзҡ„е·ҘдҪңжңәеҷЁдёҠдҝқеӯҳиҮӘе·ұзҡ„з§Ғй’ҘжқҘи§ЈеҜҶж–Үд»¶гҖӮ

  • йҖҡиҝҮgit-secretе°ҶеҢ…еҗ«ж•Ҹж„ҹдҝЎжҒҜзҡ„ж–Үд»¶ж”ҫе…Ҙ.gitignoreж–Үд»¶дёӯйҒҝе…ҚиҜҜжҸҗдәӨпјҢеҗҢж—¶з”ҹжҲҗдёҖдёӘж–°зҡ„еҠ еҜҶж–Үд»¶пјҲеңЁеҺҹжңүж–Үд»¶еҗҚдёҠж·»еҠ дәҶдёҖдёӘйўқеӨ–зҡ„еҗҺзјҖ.secretпјүе№¶дҝқеӯҳеңЁGitд»“еә“дёӯгҖӮ

  • еҪ“жҠҠд»Јз Ғд»“еә“е…ӢйҡҶеҲ°жң¬ең°пјҲеҸӘеҢ…еҗ«еёҰжңүеҗҺзјҖ.secretзҡ„еҠ еҜҶж–Үд»¶пјүеҗҺпјҢеҸҜд»ҘйҖҡиҝҮgit-secretи§ЈеҜҶеҗҺз”ҹжҲҗи§ЈеҜҶж–Үд»¶пјҲдёҚеҢ…еҗ«.secretеҗҺзјҖпјүпјҢиҝҷж ·е°ұеҸҜд»ҘеңЁжң¬ең°дҪҝз”ЁеҢ…еҗ«иҝҷдәӣж•Ҹж„ҹдҝЎжҒҜзҡ„ж–Үд»¶дәҶгҖӮ

git-secretзҡ„е®үиЈ…е’ҢдҪҝз”Ё

git-secretзҡ„дҪҝз”Ёдҫқиө–дәҺgitе’ҢGnuPGгҖӮGitзҡ„е®үиЈ…еҸҜд»ҘеҸӮиҖғзӣёе…іж–ҮжЎЈпјҢдёҚеңЁиҝҷйҮҢиөҳиҝ°гҖӮgit-secretдҪҝз”Ёзҡ„еҜҶй’Ҙд»ҘеҸҠеҠ и§ЈеҜҶзҡ„еҠҹиғҪйғҪжҳҜйҖҡиҝҮGnuPGжқҘе®ҢжҲҗзҡ„гҖӮ

  • еҲқе§ӢеҢ–дёҖдёӘж–°зҡ„Gitд»“еә“жҲ–иҖ…е…ӢйҡҶдёҖдёӘе·Із»ҸеӯҳеңЁзҡ„д»Јз Ғд»“еә“
